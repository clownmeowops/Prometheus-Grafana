# Monitoring Stack: Prometheus + Grafana

Готовый стек для мониторинга IT-инфраструктуры в Docker. Позволяет собирать системные метрики и визуализировать их в реальном времени.

## Быстрый запуск

Мы автоматизировали развертывание, чтобы вам не пришлось настраивать всё вручную.

**Windows:**
Запустите файл `setup.bat`. Он создаст необходимые директории и поднимет все контейнеры.

**Linux / macOS:**
```bash
chmod +x setup.sh
./setup.sh

    Компоненты стека

Prometheus: Система для сбора и хранения временных рядов (метрик).
Grafana: Платформа для визуализации данных и построения дашбордов.
Node Exporter: Агент для сбора аппаратных метрик (CPU, RAM, диск, сеть).

    Доступ к сервисам
    После успешного запуска перейдите по ссылкам:
Grafana: http://localhost:3000 (Login/Pass: admin / admin)
Prometheus: http://localhost:9090

    Настройка мониторинга
В Grafana перейдите в Data Sources → Add data source → Prometheus.
В поле URL укажите: http://prometheus:9090.
Импортируйте готовый дашборд с ID 1860 (Node Exporter Full).

Этот проект создан для обучения DevOps-практикам и мониторингу систем в среде Docker.
